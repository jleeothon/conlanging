== Definitions

=== witan

Means: to know (a fact). Followed by an object noun, or a relative phrase
introduced by _þat_.

Examples:

- Ik wit þet. (_I know it_)
- We weitĕm þat þe kat þen mús at. (_We know that the cat ate the mouse_).
